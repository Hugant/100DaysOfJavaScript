Создание мини викторины, не должно затруднить вас, но все же.


Для начала, нужно создать несколько вопросов.

				
html:

				<div>Если бы осьминоги умели считать, то какой бы системой они скорее всего пользовались?</div>
<div>
    <input id = "FirstQ1type = "button" value = "Двоичной"          onClick = "checkAnswer(1, 1)">
    <input id = "SecondQ1type = "button" value = "Восьмеричной"      onClick = "checkAnswer(1, 2)">
    <input id = "ThirdQ1type = "button" value = "Десятичной"        onClick = "checkAnswer(1, 3)">

<div>
    <input type = "button" value = "2"  onClick = "checkAnswer(2, 1)">
    <input type = "button" value = "3"  onClick = "checkAnswer(2, 2)">
    <input type = "button" value = "5"  onClick = "checkAnswer(2, 3)">
    <input type = "button" value = "10" onClick = "checkAnswer(2, 4)">

Теперь нужно создать функцию, которая будет говорить правильно ли мы ответили на вопрос.

Функция будет принимать два параметра: номер вопроса, номер ответа.

				
JavaScript:

				var check1 = 0;
var check2 = 0;// эти переменные отвечают за одноразовый ответ на каждый вопрос

function checkAnswer(question, answer) {
    if(question == 1) { // проверка номера вопроса
        if(check1 < 1) { // проверка отвечали ли на этот вопрос раньше
            switch(answer) {
                case 1:
                    document.getElementById("FirstQ1").style.background = "red";
                    break;
                case 2:
                    document.getElementById("SecondQ1").style.background = "#00e600";// правильный ответ
                    break;
                case 3:
                    document.getElementById("ThirdQ1").style.background = "red";
                    break;
                case 4:
                    document.getElementById("FourthQ1").style.background = "red";
                    break;
            }
            check1 = 1;// для одноразового ответа
        }
    }
}
