Random background color

Создание скрипта, который меняет цвет фона очень прост, и реализовать его сможет даже школьник!


Для начала, вам нужно создать кнопки, которые выполняют функции смена фона.

				
html:

				<input type = "button" value = "Red"    onclick = "setColor(this.value)">
<input type = "button" value = "Yellow" onclick = "setColor(this.value)">
<input type = "button" value = "Blue"   onclick = "setColor(this.value)">
<input type = "button" value = "Pink"   onclick = "setColor(this.value)">
<input type = "button" value = "Aqua"   onclick = "setColor(this.value)">
<input type = "button" value = "Orange" onclick = "setColor(this.value)">
<input type = "button" value = "Green"  onclick = "setColor(this.value)">
<input type = "button" value = "Brown"  onclick = "setColor(this.value)">
<input id = "RandButton"   type = "button" value = "Random" onclick = "randColor()">

После вы должны создать функцию, которая будет влиять на цвет фона.


С помощью следующего метода вы получаете доступ к элементу по идентификатору: 

				document.getElementById("идентификатор")

Если обратиться к свойству style предыдущего метода, тогда можно получить стили элемента :

				document.getElementById("идентификатор").style

Получение доступа к определенному стилю:

				document.getElementById("идентификатор").style.background = "blue";// фон
document.getElementById("идентификатор").style.color = "red";// цвет текста
document.getElementById("идентификатор").style.fontSize "24px";// размер текста

Полный список свойств вы можете увидеть здесь

Здесь есть несколько особенностей:

  • Значение, которое вы присваиваете должно быть строкой = "red"
  • Если свойство имеет единицу измерения px, em и т.д, тогда они должны быть приписаны в конце числа без пробела = "24px", "24em"


Чтобы получить свойства элемента body достаточно написать следующее: 

				document.body.style

Перейдем к скрипту


Функция, которая изменяет цвет фона на опеределенный цвет, очень проста.

Мы должны узнать какое значение у нашей кнопки(Red, Green, Yellow) и присвоить это значение цвету фона.

				
JavaScript:

				function setColor(color) {// В переменной color записан текущий цвет
    document.body.style.background = color;// присваиваем этот цвет
}

Вы наверное задались вопросом, что это за странная передача значения: this.value.

Ключевое слово this ссылается на родителя или глобальный объект.

В данный момент this возвращает следующее: 

				<input id="RedButton" type="button" value="Red" onclick="setColor(this.value)">

Теперь мы можем обратиться к свойству value.

Вот что возвращает this.value: 

				Red

Ситуация меняется при создании кнопки, которая задает случайный цвет фона.


Метод Math.random() возвращает от 0(включительно) до 1.

Метод Math.round() возвращает число, округленное к ближайшему целому.


Метод [число].toString([radix]) возвращает строковое представление числа

Где radix это число между 2 и 36, определяющее основание системы счисления в которой нужно представить число(Это необязательный параметр).


Так как решить эту задачу зная эти функции?

Изначально нужно задать случайные значение каждому из цветов RGB.

				
JavaScript:

				function randColor() {
    var color = Math.round(255*Math.random());// Случайное число от 0 до 255
    var R = color.toString(16);// Перевод в 16 систему счисления от 00 до FF
    color = Math.round(255*Math.random());
    var G = color.toString(16);
    color = Math.round(255*Math.random());
    var B = color.toString(16);
    color = R + G + B;// FFA2F3
    document.getElementById("body").style.background = "#" + сolor;// #FFA2F3
}
