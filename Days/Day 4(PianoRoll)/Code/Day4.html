Перейдем же к скрипту

				var a = new Audio("url");
a.play();

Создает новый элемент с типом Audio, с указанием url звукового файла в параметрах.

Метод a.play() воспроизводит звуковой файл.

				
JavaScript:

				var red = new Audio("red.wav");
var orange = new Audio("orange.wav");
var yellow = new Audio("yellow.wav");
var green = new Audio("green.wav");
var cyan= new Audio("cyan.wav");
var blue = new Audio("blue.wav");

function soundClick(button) {
    switch(button) {
	case "red":
		red.stop();// остановка
		red.play();// воспроизведение
		break;

	case "orange":
		orange.stop();
		orange.play();
		break;

	case "yellow":
		yellow.stop();
		yellow.play();
		break;

	case "green":
		green.stop();
		green.play();
		break;

	case "cyan":
		cyan.stop();
		cyan.play();
		break;

	case "blue":
		blue.stop();
		blue.play();
		break;
    }
}
// функция остановки
HTMLAudioElement.prototype.stop = function() {
	this.pause();// приостанавливаем
	this.currentTime = 0.0;// обнуляем время воспроизведения
}

У элемента Audio нет метода остановки звука, но есть метод pause(), который приостанавливает воспроизведение.

И метод currentTime, который отвечает за текущее время воспроизведения звукового файла.

Пользуясь этими методами, я написал функцию остановки воспроизведения звукового файла.
